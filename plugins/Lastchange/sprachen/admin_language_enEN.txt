
config_lastchange_plugin_name                  = Lastchange
config_lastchange_plugin_desc                  = Displays the last change.<br><br><span style="font-weight:bold;">Usage:</span><br>{Lastchange} prints something like: "Last change: Welcome (23.10.2020, 9:07:20)"<br>{Lastchange|text} prints something like: "Last change"<br>{Lastchange|page} prints something like: "Welcome"<br>{Lastchange|pagelink} prints something like: "Welcome" (with a link to the content page)<br>{Lastchange|date} prints something like: "(23.10.2020, 9:07:20)"<br><br><span style="font-weight:bold;">Configuration:</span><br>The plugin retrieves the text "Last change" and the Date format from the CMS language file; you can also specify both yourself. The date format is based on the PHP date() function.
config_lastchange_plugin_lastchange            = Last change with link and date
config_lastchange_plugin_text                  = Last change text:
config_lastchange_plugin_page                  = Name of the last changed content page
config_lastchange_plugin_pagelink              = Link to the last changed content page
config_lastchange_plugin_date                  = Date of last change
config_lastchange_messagetext                  = Custom text for last change:
config_lastchange_date                         = Custom date format:
config_lastchange_showhiddenpagesinlastchanged = Include hidden content pages?
